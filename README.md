# JQuery BlockUI

Rails wrapper for jQuery-BlockUI.

## Installation

Add this line to your application's Gemfile:

    gem 'jquery_block_ui'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install jquery_block_ui
